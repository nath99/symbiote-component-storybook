# Utlity Classes

Our range of utility classes help when positioning elements

### Margin & Padding

Adding margin or padding to elements.

```html
<div class="mt-2 py-3">
    This div has margin top 1em
</div>
```

### Positioning

Our positioning classes make it easy to position elements.

Each positioning type is prefixed with `pos-`.

```html
<div class="pos-absolute t-2 r-2">
    Absolutely positioned with top 1em and right 1em.
</div>
<div class="pos-fixed t-1 l-1">
    Fixed position with top 0.5em and left 0.5em.
</div>
```
