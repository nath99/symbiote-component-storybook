The DatePicker component is useful for easily selecting a specific date or a range of days. We have used mymths's VanillaJS Date Picker due to its clean and accessable user interface as well as its wide range of configuration options. The full documentation can be found <a href="https://mymth.github.io/vanillajs-datepicker/" target="_blank" rel="noopener noreferrer">here</a>. It is released under an MIT licence and can therefore be used for most commercial and non-commercial projects.

### When to use this component

This component should be used when users need to input a date or date-range and will benefit from the context of a calendar.

### Options

The following optional items can be supplied to the DatePicker component.

- **Type** - The main setting for either picking a specific date or a range between two dates
- **Label** - A label/title that can be assigned to either a date or date range picker
- **ID** - A parameter for setting the id for the rendered DatePicker component
- **Classname** - A parameter for setting the classname for the rendered DatePicker component
